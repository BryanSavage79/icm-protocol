# ICM Protocol Specification

This repository is the public specification for the ICM Protocol.

## Purpose

This repo defines the canonical, open standard used by all ICM-compatible clients and services. It documents protocol contracts only and does not contain implementation code.

## Protocol Scope Summary

- **Handles**: Placeholder specification for ICM handle format and validation rules.
- **Harmonic Hex Strings**: Placeholder specification for harmonic hex encoding and decoding requirements.
- **Identity Flow**: Placeholder specification for identity resolution, verification, and role adaptation lifecycle.
- **Scene Manifests**: Placeholder specification for scene manifest structure, constraints, and versioning.
- **Data Vectors**: Placeholder specification for data vector schemas, transport, and semantic guarantees.
- **Webhooks**: Placeholder specification for webhook event contracts, delivery expectations, and signatures.

## Repository Layout

- `api/rest/`: OpenAPI placeholder entries for REST protocol surfaces.
- `api/graphql/`: GraphQL schema placeholder for protocol queries/mutations/subscriptions.
- `models/`: Canonical JSON schema placeholders for core protocol entities.
- `examples/`: End-to-end placeholder payload flows by role.
- `webhooks/`: Event payload placeholder definitions.
- `CHANGELOG.md`: Specification change history.
